Introduction to texty-ps

FTD texty-ps helps you to create your personal blog or site with ease. Perfect for the personal site of techies. This ps(Personal site) template uses Roboto-mono monospaced font from Roboto font family. Its unique look and feel attracts tech gigs.

FTD ps can be applied to your websites easily. We recommend you first go through our FASTN template documentation.

Create your first FASTN project GitHub repository using our FASTN template.

Now follow below given steps to start using this texty-ps template:

Add below lines to FPM.ftd file:
null
-- fastn.dependency: fifthtry.github.io/texty-ps as ps
-- fastn.auto-import: ps
